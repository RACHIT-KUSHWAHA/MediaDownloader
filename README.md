## ✨ Features
- ⚡ **Asynchronous:** Built with Pyrogram for concurrent request handling.
- 📦 **Dockerized:** Runs in an isolated container for stability.
- ☁️ **Cloud Hosted:** Deployed on **Microsoft Azure** (Southeast Asia).
- 🎥 **Multi-Platform:** Supports Instagram Reels and YouTube videos.

## ⚙️ Deployment
This bot is deployed on an **Ubuntu 24.04 LTS** instance on **Azure**. 
- **Instance Size:** Standard B2ats_v2 (1 GiB Memory).
- **Container:** Docker managed.
